# Android App Preview

Puede abrir este enlace en su teléfono [https://app.apphive.io](https://app.apphive.io) o escanear este código QR

![Apphive Previewer App](../.gitbook/assets/descarga-1.png)

{% hint style="info" %}

{% endhint %}
